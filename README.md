# ClinicalMed

Sistema de gerenciamento de senhas para clínicas.

## Sobre o projeto

Aplicação web desenvolvida para simular um sistema de atendimento em clínicas, permitindo gerar senhas por categoria e organizar a fila de pacientes.

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## Funcionalidades

- Geração de senhas
- Categorias de atendimento:
  - Consulta
  - Exames
  - Preferencial
  - Retorno
- Controle de fila

## Próximas melhorias

- Banco de dados
- Sistema de login
- Painel de chamada
- Integração com servidor
